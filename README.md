# PROMIR-BUD — Modular Homes & Social Containers

A static website for PROMIR-BUD, a modular construction company based in Starachowice, Poland. Features modular homes, social containers, and skeletal modules with turnkey construction services.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
